# 📱 JodNid (จดนิด) — ผู้ช่วยบันทึกการเงินส่วนบุคคลอัจฉริยะผ่าน LINE

**JodNid (จดนิด)** คือระบบผู้ช่วยส่วนตัวบนแอปพลิเคชัน LINE ที่ช่วยให้ผู้ใช้งานสามารถบันทึกรายรับ-รายจ่ายของตัวเองได้ง่ายที่สุด ผ่านการคุยแชทปกติ โดยระบบจะนำเทคโนโลยี AI และระบบตรวจจับอัจฉริยะมาช่วยคิด วิเคราะห์ และจัดหมวดหมู่ให้โดยอัตโนมัติ เพื่อตัดความยุ่งยากในการที่ผู้ใช้ต้องมานั่งกรอกข้อมูลเองทีละช่อง

---

## 🚀 ภาพรวมขั้นตอนการทำงานของระบบ (How It Works)

การทำงานของ JodNid ถูกออกแบบมาให้สั้น กระชับ และจบไวภายในหน้าแชท โดยแบ่งออกเป็น 4 ขั้นตอนหลัก ดังนี้:

### 1. รับข้อมูลจากผู้ใช้งาน (User Input)
ผู้ใช้งานสามารถส่งข้อมูลการใช้เงินเข้ามาในห้องแชทของ LINE ได้ 2 รูปแบบหลักๆ คือ:
* **พิมพ์ข้อความธรรมดา:** เช่น *"กินข้าวหมูกรอบไป 80 บาท"*, *"ค่ากาแฟ 60"* หรือ *"เติมน้ำมัน 1,000"*
* **ส่งรูปภาพสลิป/ใบเสร็จ:** เช่น รูปภาพสลิปโอนเงินธนาคาร หรือภาพถ่ายใบเสร็จจากร้านค้า

### 2. วิเคราะห์และสกัดข้อมูลอัจฉริยะ (Intelligent Parsing)
เมื่อระบบได้รับข้อมูล จะทำการประมวลผลทันทีเพื่อหาคำตอบ 3 อย่าง:
* **จำนวนเงินเท่าไหร่?** (Amount)
* **ซื้อรายการอะไรไป?** (Item Name)
* **ควรอยู่หมวดหมู่ไหน?** (Category) ตัวอย่างเช่น หากพิมพ์ว่า "กะเพราไข่ดาว" ระบบจะรู้ทันทีว่าต้องจัดอยู่ในหมวด **"อาหาร"**

> 💡 **ฟีเจอร์เด่นเรื่องความเร็ว:** ระบบมีโหมดการทำงานร่วมกัน หากผู้ใช้พิมพ์ข้อความสั้นๆ ทั่วไป ระบบจะสแกนคำและตอบกลับทันทีในเสี้ยววินาที แต่หากข้อความมีความซับซ้อนหรือเป็นรูปภาพ ระบบจะส่งต่อให้ AI ช่วยคิดวิเคราะห์ เพื่อให้การจัดหมวดหมู่แม่นยำที่สุดโดยที่ผู้ใช้ไม่ต้องเหนื่อยเลือกเอง

### 3. ยืนยันรายการและตัดงบประมาณ (Confirmation & Budgeting)
* **โหมดปกติ:** ระบบจะส่ง "บิลสรุปรายการชั่วคราว" หน้าตาสวยงามกลับไปให้ผู้ใช้กดปุ่มตรวจสอบความถูกต้องก่อน หากถูกต้องก็กด "ยืนยัน" เพื่อบันทึกจริง
* **โหมดบันทึกด่วน (Bypass Mode):** สำหรับผู้ที่ชอบความรวดเร็ว สามารถเปิดโหมดนี้เพื่อให้ระบบบันทึกเงินลงฐานข้อมูลทันทีหลังจากส่งข้อความ โดยไม่ต้องกดปุ่มยืนยันซ้ำอีกรอบ
* **การตัดงบ:** หลังจากยืนยัน รายการเงินจะวิ่งไปหักลบออกจาก **"งบประมาณรายเดือน (Budget)"** ของหมวดหมู่นั้นๆ ทันที

### 4. สรุปผลลัพธ์ (Financial Summary)
หลังจากบันทึกเสร็จสิ้น ระบบจะส่งข้อความสรุปกลับไปแจ้งเตือนผู้ใช้ทันที เพื่อให้ทราบว่า:
* บันทึกรายการอะไร สำเร็จแล้วบ้าง
* ยอดเงินคงเหลือในงบประมาณของหมวดหมู่นั้นๆ ในเดือนนี้เหลืออยู่กี่บาท

## 💎 จุดเด่นของโปรเจกต์ที่มอบให้กับผู้ใช้งาน

* **Zero-Click Experience:** พิมพ์ครั้งเดียวจบ AI คิดให้เสร็จสรรพ ไม่ต้องกดเลือกหมวดหมู่หลายๆ ครั้งให้เสียเวลา
* **คุ้นเคยและใช้งานง่าย:** ทำงานอยู่บนแอปพลิเคชัน LINE ที่ทุกคนมีติดเครื่องอยู่แล้ว ไม่ต้องดาวน์โหลดแอปพลิเคชันอื่นเพิ่ม
* **ควบคุมค่าใช้จ่ายได้จริง:** มีระบบแจ้งเตือนงบประมาณคงเหลือแบบเรียลไทม์ทุกครั้งที่มีการบันทึก ช่วยให้ผู้ใช้ยับยั้งชั่งใจก่อนใช้เงินเกินตัวได้ดีขึ้น
